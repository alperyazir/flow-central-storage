# Story 11.2: Layout & Navigation - Collapsible Sidebar

## Status
Done

## Story
**As an** admin user,
**I want** a modern collapsible sidebar navigation layout,
**so that** I can navigate the admin panel efficiently with a clean, spacious dashboard interface.

## Acceptance Criteria
1. Collapsible sidebar component using shadcn/ui Sidebar pattern (Radix-based)
2. Sidebar expands (16rem) and collapses (icon-only ~3rem) with toggle button
3. All existing navigation links preserved with Lucide icons (Dashboard, Publishers, All Books, AI Processing, AI Settings, AI Data Viewer, Applications, Bundles, Teachers, Trash)
4. Brand name "Flow Central Storage" shown when expanded, logo/icon when collapsed
5. Theme toggle (light/dark) in sidebar footer
6. User menu with logout in sidebar footer
7. Sticky header bar with sidebar trigger (hamburger) button
8. Mobile responsive: sidebar as overlay/sheet on screens < 768px
9. Main content area fills remaining space with proper padding (p-6 md:p-8)
10. Active route highlighted in sidebar

## Dev Notes

### Sidebar Architecture
Follow the shadcn/ui sidebar pattern:
```
SidebarProvider (context for open/collapsed state)
├── AppSidebar
│   ├── SidebarHeader (brand)
│   ├── SidebarContent (nav menu)
│   └── SidebarFooter (theme + user)
└── SidebarInset
    ├── header (sticky, with SidebarTrigger)
    └── main (Outlet)
```
