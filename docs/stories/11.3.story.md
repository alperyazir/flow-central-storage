## Dev Notes

### Login Page Structure
```
<div class="flex min-h-screen items-center justify-center bg-muted">
  <Card class="w-full max-w-sm">
    <CardHeader>
      <CardTitle>Flow Central Storage</CardTitle>
      <CardDescription>Sign in to your account</CardDescription>
    </CardHeader>
    <CardContent>
      <form>
        <Label + Input (email)>
        <Label + Input (password)>
        <Alert (error, conditional)>
        <Button type="submit">
      </form>
    </CardContent>
  </Card>
</div>
```

### Auth Store Integration
- Uses `useAuthStore` - no changes needed to the store itself
- `login(email, password)` → sets token + redirects
- `isAuthenticated` → guards protected routes
- `isHydrating` → shows spinner while checking stored session

- Story 11.1 (foundation) and 11.2 (layout) must be complete
- Button component from 11.2 is reused here
